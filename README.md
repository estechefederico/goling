# Goling – Landing Dropshipping en Argentina

Sitio de **Goling**: landing de dropshipping en Argentina. Ofrece tienda online en 7 días (planes Starter y Premium), sin stock propio, con Tiendanube y proveedores locales.

**Dominio:** [dropshipping.goling.com.ar](https://dropshipping.goling.com.ar)

**Estructura:** El sitio vive en `view/` (index.html, css/, js/, images/). En la raíz solo está `index.html` para GitHub Pages, con rutas que apuntan a `view/` (view/css/, view/js/, view/images/). Si editás `view/index.html`, copiá su contenido a `index.html` en la raíz y reemplazá en la raíz: `href="css/` → `href="view/css/`, `href="images/` → `href="view/images/`, `src="js/` → `src="view/js/`, `src="images/` → `src="view/images/`.
